## csv

`Ex csv.writer` 

```
import csv
with open('lunch.csv', 'w', encoding = 'utf-8') as f:
    csv.writer = csv.writer(f)
    for item in lunch.items():
        csv.writer.writerow(item)
```

`Ex csv.reader`

```
import csv
with open('lunch.csv', 'r', encoding = 'utf-8') as f:
    items = csv.reader(f)
    for item in items:
        print(item)
```
